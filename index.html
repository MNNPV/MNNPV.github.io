Protocol: Beginner Onboarding

Standard Operating Procedure for new research assistants joining the NexPOS project.

  • Repository Configuration

    Clone the GitHub repository. Ensure Node.js v18+ is installed. Run npm install to fetch dependencies. Do not commit `.env` files.

  • Literature Review Methodology

    Access the shared Zotero library. All new citations must be tagged with the relevant architectural pattern (e.g., 'MVC', 'Event-Sourcing').

  • Code Style Guidelines

    We strictly follow the Airbnb Style Guide for JavaScript. Run linter before pushing any commit. Commit messages must follow Conventional Commits (e.g., feat: add login screen).

  • Agile Workflow

    Tasks are tracked on the board below. Pick items from the 'Backlog'. Move to 'In Progress' only when actively working.
