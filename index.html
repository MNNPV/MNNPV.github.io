POS System Dissertation

An academic exploration into modern Point of Sale systems, integrating mobile and web application architectures. This project aims to standardize the user experience across devices without cognitive load.

Problem Statement

Current POS solutions often suffer from bloated interfaces and lack of true cross-platform responsiveness.

Objectives

Develop a "No-Think" standard interface that adapts seamlessly to any device size without gradient clutter.

Methodology

Mixed-methods research utilizing A/B testing on UI prototypes and qualitative surveys on usability.

Brand Guidelines

Strict adherence to flat, functional design. No icons. No gradients. Focus on typography and whitespace.
