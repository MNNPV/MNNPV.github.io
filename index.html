Project Abstract

Current retail POS systems suffer from latency issues during network outages and lack predictive analytics for inventory management. This project proposes "NexPOS," a hybrid-cloud solution utilizing PouchDB for local storage with eventual consistency synchronization to a MongoDB cloud cluster. Furthermore, we aim to integrate a TensorFlow-based prediction model to suggest re-stocking levels based on historical sales data.
